## 📋 Description
ChatApp est une application de messagerie instantanée développée dans le cadre d'un apprentissage de Node.js et MongoDB. Elle permet aux utilisateurs de communiquer en temps réel via des messages texte, audio, vidéo et fichiers, à l'instar de WhatsApp.

## ✨ Fonctionnalités
Authentification sécurisée : Inscription et connexion avec numéro de téléphone

Messagerie en temps réel : Échange de messages instantanés via WebSocket (Socket.io)

# Types de messages :

- 📝 Messages texte
- 🎵 Messages audio (à venir)
- 🎥 Messages vidéo (à venir)
- 📎 Fichiers (documents, images, etc.)-à venir

Gestion des conversations : Liste des discussions avec dernier message

Statuts des messages : ✓ (envoyé), ✓✓ (délivré), ✓✓ bleu (lu)

Recherche d'utilisateurs : Par numéro de téléphone

Interface responsive : Design adapté aux mobiles et desktop
